## 1. 课题来源、目的及意义

### 1.1 课题来源

本课题来源于医学影像智能诊断领域的实际需求。乳腺癌是全球女性最常见的恶性肿瘤之一，早期发现和准确诊断对于提高患者生存率具有重要意义。超声检查因其无创、无辐射、成本低廉等优势，已成为乳腺癌筛查的重要手段。然而，传统的超声图像诊断高度依赖医生的专业经验，存在主观性强、效率较低、漏诊率较高等问题。近年来，深度学习技术在医学图像分析领域取得了突破性进展，为实现乳腺癌超声图像的自动化、智能化诊断提供了新的技术途径。

### 1.2 课题目的及意义

本课题旨在利用深度学习技术，构建一个具有可解释性的乳腺癌超声图像自动诊断系统。该系统能够自动对乳腺超声图像进行分类（区分正常、良性、恶性）和病灶区域分割，并通过热力图可视化技术展示模型的决策依据，辅助医生进行诊断。

**理论意义**：本研究将深入探索卷积神经网络（CNN）和Transformer等深度学习模型在医学图像分析中的应用，对比分析不同网络架构的性能特点，为医学影像智能诊断领域提供技术参考和理论支撑。

**实践意义**：本系统可作为医生的辅助诊断工具，提高乳腺癌筛查的效率和准确性，减少漏诊和误诊，具有重要的临床应用价值。同时，系统的可解释性设计有助于增强医生对AI诊断结果的信任度，促进人工智能技术在医疗领域的落地应用。

## 2. 国内外研究现况及发展趋势

### 2.1 深度学习在医学图像分析中的应用现状

深度学习，特别是卷积神经网络（CNN），已在医学图像分析领域取得了显著成果。简单来说，深度学习是一种让计算机通过大量数据自动学习特征和规律的技术，就像人类通过观察大量案例来积累经验一样。CNN是专门用于处理图像的深度学习模型，它能够自动从图像中提取有用的特征信息。

在乳腺超声图像分析方面，研究者们已经探索了多种深度学习方法。**ResNet**（残差网络）通过引入"跳跃连接"解决了深层网络训练困难的问题，使得网络可以更深、性能更强。**EfficientNet**则通过平衡网络的深度、宽度和分辨率，在保持较少参数量的同时实现了优异的性能。近年来，**Transformer**架构从自然语言处理领域引入到计算机视觉中，其核心的"自注意力机制"能够捕捉图像中远距离像素之间的关系，在医学图像分析中展现出良好的潜力。

### 2.2 图像分割技术的发展

图像分割是指将图像中的特定目标区域（如病灶）从背景中分离出来的技术。**U-Net**是医学图像分割领域最具影响力的模型之一，其采用对称的"编码器-解码器"结构：编码器负责提取图像特征并逐步降低分辨率，解码器则负责恢复分辨率并生成分割结果。U-Net的"跳跃连接"设计能够融合不同层次的特征信息，非常适合医学图像中小目标的精确分割。

在U-Net基础上，研究者们提出了多种改进变体。**Attention U-Net**引入注意力机制，使模型能够自动关注重要区域；**TransUNet**则将Transformer与U-Net结合，利用Transformer的全局建模能力提升分割性能。

### 2.3 可解释性技术的研究进展

深度学习模型通常被视为"黑盒"，即我们难以理解模型为什么会做出特定的预测。在医疗领域，可解释性尤为重要，因为医生需要理解AI的诊断依据才能做出最终决策。**Grad-CAM**（梯度加权类激活映射）是一种广泛使用的可解释性技术，它通过分析模型内部的梯度信息，生成热力图来显示模型在做出预测时关注的图像区域。热力图中颜色越红的区域表示模型越关注，这有助于医生判断AI的诊断是否合理。

### 2.4 发展趋势

当前，医学图像智能诊断领域呈现以下发展趋势：一是多任务联合学习，即同时进行分类和分割任务以提升整体性能；二是轻量化模型设计，以适应临床实际部署需求；三是可解释性与可信AI研究，增强模型的透明度和可靠性；四是多模态数据融合，结合超声、病理等多种数据源进行综合诊断。

## 3. 课题研究的内容和技术方案

### 3.1 课题研究内容

本课题的研究内容主要包括以下三个方面：

**（1）乳腺超声图像分类**：构建深度学习分类模型，实现对乳腺超声图像的三分类（正常、良性、恶性）。对比CNN（如ResNet）与Transformer等不同架构的性能表现。

**（2）病灶区域分割**：利用U-Net系列模型实现乳腺病灶区域的精确分割，为医生提供病灶的位置和范围信息。

**（3）可解释性诊断辅助**：引入Grad-CAM技术生成热力图，可视化模型关注的病灶区域，增强诊断结果的可解释性和可信度。

### 3.2 课题预期目标

|指标类型|具体指标|预期值|
|:--|:--|:--|
|分类性能|AUC（曲线下面积）|>0.85|
|分类性能|F1-Score|>0.80|
|分割性能|Dice系数|>0.75|
|分割性能|IoU（交并比）|>0.65|
|系统功能|热力图可视化|支持|

### 3.3 技术路线与方案

#### 3.3.1 关键技术

**（1）数据预处理与增强**：对BUSI数据集进行标准化处理，包括图像尺寸统一（224×224像素）、灰度归一化等。采用数据增强技术（水平/垂直翻转、随机旋转、亮度对比度调整、高斯噪声）扩充训练样本，提升模型的泛化能力。针对数据集中类别不平衡问题，采用分层采样策略确保各类别样本的均衡。

**（2）迁移学习**：迁移学习是指将在大规模数据集（如ImageNet）上预训练好的模型应用到新任务上的技术。由于医学图像数据通常较少，直接从头训练深度模型容易过拟合。通过加载预训练权重，模型已经具备了提取通用图像特征的能力，只需在目标数据集上进行微调即可获得良好性能。

**（3）损失函数设计**：分类任务采用交叉熵损失函数；分割任务采用Dice损失与二元交叉熵损失的组合，以应对医学图像分割中前景与背景面积不平衡的问题。

**（4）模型评估方法**：采用分层K折交叉验证（K=5）确保评估结果的可靠性，计算各项指标的均值和标准差，并进行统计显著性分析。

#### 3.3.2 总体方案

本系统采用模块化设计，总体架构分为四个核心模块：

**系统总体架构说明**：

```
┌─────────────────────────────────────────────────────────────────┐
│                        系统总体架构                              │
├─────────────────────────────────────────────────────────────────┤
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────────────┐  │
│  │ 数据处理模块 │───→│ 模型推理模块 │───→│ 可视化与输出模块    │  │
│  └─────────────┘    └─────────────┘    └─────────────────────┘  │
│        │                  │                      │              │
│        ▼                  ▼                      ▼              │
│  ·图像预处理        ·分类子模块           ·热力图叠加显示        │
│  ·数据增强          (ResNet18)           ·分割轮廓绘制          │
│  ·数据集划分        ·分割子模块           ·诊断报告生成          │
│                     (U-Net)                                     │
│                     ·Grad-CAM模块                               │
└─────────────────────────────────────────────────────────────────┘
```

**模块功能详述**：

|模块名称|主要功能|技术实现|
|:--|:--|:--|
|数据处理模块|图像读取、预处理、数据增强、数据集划分|OpenCV、Albumentations|
|分类子模块|乳腺超声图像三分类|ResNet18+迁移学习|
|分割子模块|病灶区域分割|U-Net(4层编码-解码结构)|
|可解释性模块|生成类激活热力图|Grad-CAM|
|可视化输出模块|结果展示与诊断报告|Matplotlib|

**数据流程**：输入乳腺超声图像→数据预处理（尺寸调整、归一化）→分类模型预测类别→分割模型生成病灶掩模→Grad-CAM生成热力图→综合输出诊断结果。

### 3.4 实验数据集

本研究使用BUSI（Breast Ultrasound Images）公开数据集，该数据集包含三类乳腺超声图像及对应的分割标注：

|类别|含义|样本特征|
|:--|:--|:--|
|Normal|正常|无明显病灶|
|Benign|良性|存在良性肿块，边界清晰|
|Malignant|恶性|存在恶性肿块，边界不规则|

数据集将按照7:1.5:1.5的比例划分为训练集、验证集和测试集，确保各子集中类别分布一致。

## 4. 可行性与风险分析

### 4.1 技术可行性

**（1）成熟的技术基础**：深度学习框架PyTorch提供了完善的模型构建和训练工具，ResNet、U-Net等经典模型已有成熟的开源实现，Grad-CAM也有现成的Python库支持。

**（2）硬件环境支持**：本项目基于Mac M系列芯片开发，PyTorch已支持MPS（Metal Performance Shaders）后端加速，能够满足模型训练需求。

**（3）数据集可获得性**：BUSI数据集为公开数据集，可免费获取用于学术研究，数据量适中，适合作为毕业设计的实验数据。

### 4.2 制约因素与风险

|风险类型|具体描述|应对措施|
|:--|:--|:--|
|数据风险|数据集规模有限，可能导致过拟合|采用数据增强、迁移学习、正则化技术|
|技术风险|模型性能可能未达预期|选用经过验证的经典模型，设置合理的性能基线|
|时间风险|开发周期较紧|制定详细的时间计划，优先完成核心功能|
|伦理风险|医学数据涉及隐私|使用公开脱敏数据集，遵守数据使用规范|

### 4.3 成本估算与可行性评估

本项目为软件开发类毕业设计，主要成本为开发时间成本。采用简化的COCOMO模型进行估算：预计代码规模约2000行（2KLOC），属于小型项目，开发工作量约为2-3人月，在毕业设计周期内完全可行。硬件方面仅需一台配备M系列芯片的Mac电脑，无需额外硬件投入。

综合以上分析，本课题在技术上可行，风险可控，具备按期完成的条件。
