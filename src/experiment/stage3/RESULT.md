# 阶段3实验结果

## 实验目标

使用DAGAN（Data Augmentation GAN）生成合成图像，扩充训练集，目标准确率 **85%**

## 实验配置

### 数据集
- 原始训练集：546张（70%）
- DAGAN合成图像：2730张（每张训练图5个变体）
- 合并后训练集：**3276张**
- 验证集：117张（15%）
- 测试集：117张（15%）

### 模型
- 架构：ResNet-18（预训练）
- 优化器：Adam（lr=0.0001, weight_decay=1e-4）
- 学习率调度：StepLR（step_size=20, gamma=0.1）
- 损失函数：CrossEntropyLoss
- 早停：patience=10

### DAGAN训练
- 训练轮数：100 epochs
- 批次大小：16
- 学习率：0.0002
- 训练时间：**50.9分钟**
- 最终损失：G_loss=3.57, D_loss=0.0018
- 重建误差：0.0257

## 技术要点

### DAGAN训练
- **只使用训练集**训练DAGAN（避免数据泄露）
- 生成器架构：U-Net + ResNet块
- 判别器架构：PatchGAN
- 损失函数：对抗损失 + 重建损失

### 数据加载
- 原始训练集 + 合成图像 = 增强训练集
- 验证集和测试集**不使用合成图像**
- 训练集应用**传统数据增强**（翻转、旋转、裁剪等）

### 防止过拟合
- 早停机制（patience=10）
- Dropout（0.5）
- 数据增强
- 预训练权重
